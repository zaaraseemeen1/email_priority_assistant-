# email_priority_assistant-
Email Priority Assistant is a full-stack application that automatically reads emails, uses AI to classify each one by priority, and displays the results in a clean web dashboard. The problem it solves: busy professionals receive hundreds of emails every day. Most are newsletters, research notes, or automated notifications that don't need attention.  This system uses a real AI model running on AWS to separate the two automatically — so the
user opens their dashboard and immediately sees what matters.
